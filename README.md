//Simulador de Imposto de Renda Pessoa Física (IRPF)

Projeto desenvolvido com foco em prática real de desenvolvimento web, organização de código, versionamento e evolução incremental.

O simulador tem como objetivo fornecer uma estimativa simplificada de imposto de renda, ao mesmo tempo em que serve como ambiente de estudo aplicado em front-end, lógica de negócio e integração de tecnologias.

O desenvolvimento segue uma abordagem progressiva, priorizando base sólida antes da implementação de funcionalidades avançadas.

//Objetivo do Projeto

Desenvolver uma aplicação web capaz de:

Receber dados de renda anual

Considerar número de dependentes

Processar uma estimativa de IRPF

Exibir resultados de forma clara e objetiva


//Além da funcionalidade final, o projeto tem foco em:

Boas práticas de estruturação

Documentação técnica
